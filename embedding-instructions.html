How to Embed the Bani Meme Generator

Important: Before using any of these embedding methods, make sure to:
  1. Deploy your app on Replit by clicking the "Deploy" button at the top of your Replit page
  2. Replace "YOUR_APP_URL" in all code examples with your actual deployed URL

Method 1: Simple iframe Embedding

This is the easiest way to embed the Bani Meme Generator on any website.

Basic Embed Code

<iframe src="YOUR_APP_URL" width="100%" height="900" style="border: none; max-width: 800px; display: block; margin: 0 auto; border-radius: 8px; box-shadow: 0 4px 15px rgba(0, 0, 0, 0.1);" title="Bani Meme Generator"></iframe>

With Container Div

<div style="width: 100%; max-width: 800px; margin: 0 auto; border-radius: 8px; overflow: hidden; box-shadow: 0 4px 15px rgba(0, 0, 0, 0.1);"> <iframe src="YOUR_APP_URL" style="width: 100%; height: 900px; border: none;" title="Bani Meme Generator"></iframe> </div>

Customization Options

  • Adjust the height value to fit your desired size
  • Modify the max-width to change the width of the generator
  • Change or remove the box-shadow and border-radius styling as needed

Method 2: Responsive Embedding

This method ensures the generator maintains proper proportions on different screen sizes.

<style> .meme-generator-container { position: relative; width: 100%; max-width: 800px; margin: 0 auto; padding-top: 125%; /* Aspect ratio: adjust as needed */ border-radius: 8px; overflow: hidden; box-shadow: 0 4px 15px rgba(0, 0, 0, 0.1); } .meme-generator-container iframe { position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none; } </style> <div class="meme-generator-container"> <iframe src="YOUR_APP_URL" title="Bani Meme Generator"></iframe> </div>

Note: The padding-top value controls the aspect ratio. Adjust it based on your generator's layout: 100% for a square, 56.25% for a 16:9 aspect ratio, etc.

Method 3: Web Component Integration

For advanced integration, you can use the Bani Meme Generator as a custom HTML element.

Step 1: Add the script

<script src="YOUR_APP_URL/meme-generator.js"></script>

Step 2: Use the custom element

<bani-meme-generator width="800" height="900" app-url="YOUR_APP_URL" ></bani-meme-generator>

Optional: Configure with attributes

<bani-meme-generator width="800" height="900" app-url="YOUR_APP_URL" initial-background="background1" initial-pose="pose2" initial-charm="charm3" top-text="TOP TEXT" bottom-text="BOTTOM TEXT" ></bani-meme-generator>

Advanced: Interact with the component via JavaScript

<script> // Get reference to the component const generator = document.querySelector('bani-meme-generator'); // Listen for events generator.addEventListener('image-generated', (e) => { console.log('Image URL:', e.detail.imageUrl); // You can use the image URL to display or save the generated image }); // Set text programmatically document.getElementById('update-button').addEventListener('click', () => { const topText = document.getElementById('top-text-input').value; const bottomText = document.getElementById('bottom-text-input').value; generator.setText(topText, bottomText); }); // Trigger download programmatically document.getElementById('download-button').addEventListener('click', () => { generator.download(); }); </script>

Note: The Web Component integration requires deploying the meme-generator.js file to your server. For Replit deployment, you'll need to add this file to your public directory.

Deployment Tips for Replit

To make your Bani Meme Generator available for embedding:

  1. Click the Deploy button in your Replit project
  2. After successful deployment, you'll receive a URL like https://your-project-name.username.repl.co
  3. Use this URL to replace YOUR_APP_URL in the embedding code
  4. For the Web Component method, upload the meme-generator.js file to your Replit public directory

Performance Tip: For better performance with embeds, consider optimizing your images and minimizing unnecessary dependencies in your app.
